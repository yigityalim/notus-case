## Kurulum

- Projeyi klonlayın

  ```bash
  # Projeyi klonlayın
  git clone https://github.com/yigityalim/notus-case.git
  ```

- Paketleri yükleyin

  ```bash
    # Proje dizinine gidin
    cd notus-case

    # Paketleri yükleyin
    pnpm install
  ```

- Projeyi çalıştırın

  ```bash
    # Geliştirme modunda çalıştırın
    pnpm dev

    # Üretim modunda çalıştırın
    pnpm build
  ```
